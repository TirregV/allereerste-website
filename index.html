Ready, set, speel!

Sid Meier Civilization V is een videospel over de geschiedenis van de mensheid. De speler kiest een beschaving met bijhorende leider om de tand des tijds te doorstaan. Zo komt ook Vader des Vaderlands Willem van Oranje voor. Maar hoe gaat een Amerikaans spel om met de Nederlandse geschiedenis? Dit project gaat terug naar onze Nederlandsche roots, naar de fundamentele vraag: 'Wie zijn wij?'

Deze experimentele website gaat in op de verschillende speelaspecten van Civilization met betrekking op Nederland. De speler creëert in de game zijn of haar eigen narratief, maar is gebonden aan de regels van het spel. Forceert het spel een bepaalde speelwijze voor Nederland zodat het 'echte' verleden de waarheid wordt? Of biedt het ruimte voor vele alternatieve feiten?

Maar met een klein beetje kennis van HTML en CSS kun je ’t ook helemaal zelf doen!

Het ontwerpen en maken van een complexe interactieve website is niet eenvoudig maar de basisregels van de talen HTML en CSS, waaruit elke webpagina is opgebouwd, heb je zo onder de knie.

Benodigdheden

  1. Een GitHub-account;
  2. GitHub Desktop;
  3. en een goede teksteditor, bijvoorbeeld Sublime Text.

GitHub is een website waarop software geplaatst kan worden, en gedeeld met andere ontwikkelaars. (GitHub is gebaseerd op het versiebeheersysteem Git, een populaire manier om gestructureerd met verschillende mensen tegelijk aan dezelfde software te werken.)

Op GitHub kun je overzichtelijk softwareprojecten bekijken en doorzoeken, maar met GitHub kun je ook gratis je eigen website hosten (en zelfs koppelen aan je eigen domeinnaam).

Aan de slag

  1. Bekijk allereerst de broncode van deze website: github.com/bertspaan/allereerste-website.
  2. Fork het project, en kopiëer de broncode naar je eigen account.
  3. Zet GitHub Pages aan! Hiermee kun je GitHub-projecten publiceren op het internet. Als dit gelukt is is je website te vinden op <gebruikersnaam>.github.io/<projectnaam>.

Op GitHub Guides is heel veel goede documentatie over GitHub te vinden.

HTML & CSS

Als je kort het HTML- en CSS-bestand waar deze website uit bestaat hebt bekeken heb je al een aardig idee hoe de broncode van een website eruit ziet. In HTML staat de structuur en de inhoud, in CSS bepaal de je stijl (kleur, lettertypes, etc.).

Er zijn heel veel goede (en vaak gratis) online cursussen, ik raad jullie aan om een van de volgende te proberen (of zelf een te zoeken):
